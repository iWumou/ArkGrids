# 方舟格 Ark Grids
一款基于 Flutter 开发的高颜值本地效率工具，集成任务全生命周期管理 + 公历农历双模式纪念日记录，数据本地安全存储，无网络隐私泄露，轻量化无广告，适配全移动端平台。

## ✨ 项目简介
Ark Grids（方舟格）专注个人日常规划与重要时刻记录：任务分类流转清晰高效，纪念日支持农历/公历与周期重复，UI 现代化渐变设计搭配全套交互动画，极简易用，守护你的生活节奏与重要回忆。

## 📦 技术栈
- 框架：Flutter >= 3.10.0
- 语言：Dart >= 3.0.0
- 本地存储：`shared_preferences`
- 农历支持：`lunar` 农历转换插件
- 平台兼容：Android / iOS

## 🚀 快速运行
### 环境准备
安装 Flutter 开发环境并配置模拟器/真机调试设备

### 依赖安装
```bash
flutter clean # 清理缓存
cd android && ./gradlew clean && cd .. # 清理 Android 缓存
flutter pub run flutter_launcher_icons # 生成图标
flutter pub get # 安装依赖
```

### 项目调试运行
```bash
flutter run
```

### 打包编译
Android 正式打包：
```bash
flutter build apk
```

## 🎯 核心功能详解
### 一、任务计划管理模块
1. 三大任务分类看板
- 进行中：新建、待执行的全部计划任务
- 已完成：标记完结的任务归档存放，支持一键撤回恢复
- 回收站：临时删除任务储存，支持恢复与永久彻底删除

2. 任务编辑能力
- 自定义标题 + 详细备注描述
- 精准日期、时间双选择器设置截止时间
- 三级优先级标签：紧急（红色）/一般（橙色）/不急（绿色）
- 自动倒计时显示：已过期/今日剩余天数智能标识

3. 交互体验
- 列表长按拖拽自由排序
- 双击唤起快捷操作菜单：编辑 / 移入回收站
- 全局动画过渡：加载动画、删除渐隐、选中缩放动效
- 空状态友好占位提示界面

### 二、纪念日记录模块
1. 双历法支持
- 公历 / 农历一键切换选择日期
- 农历生日、传统节日完美适配换算

2. 周期重复规则
- 不重复 / 每周重复 / 每月重复 / 每年重复 四种模式
- 自动计算下一次纪念日时间，智能倒计时更新

3. 数据与交互
- 自定义纪念日名称备注
- 双击唤起编辑、删除操作面板
- 列表拖拽排序，自由调整展示顺序
- 自动天数计算：今天/明天/剩余天数/已过天数动态展示

## 🎨 UI 设计特色
- 全局渐变柔和配色主题，主色调 #667EEA
- 圆角卡片式布局，轻量化阴影分层设计
- 弹窗、底部菜单全套定制美化样式
- 空页面插画占位，交互细节打磨完善
- 按钮、标签、选择器统一视觉规范

## 🔒 隐私安全说明
1. 所有任务、纪念日数据仅通过 `SharedPreferences` 本地存储
2. 无网络请求、无云端同步、无隐私数据上传
3. 纯离线应用，杜绝个人信息泄露风险

## 📄 目录简要说明
- `PlanPage`：全功能任务管理核心页面
- `MemorialDayPage`：公历农历纪念日管理核心页面
- 全局封装通用组件：按钮、标签、弹窗、日期选择器
- 工具方法：倒计时计算、农历公历转换、优先级配色管理

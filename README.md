# 方舟格 (ArkGrids)

一个高效的任务与纪念日管理工具，旨在帮助用户更好地规划生活与记录重要时刻。

## 功能特性

### 1. 计划功能
通过三个核心箱体管理任务流程：
- **收集箱**：快速捕获所有待办事项与灵感。
- **收货箱**：整理已分类的任务，准备执行。
- **垃圾箱**：归档已完成或不再需要的任务。

### 2. 纪念日功能
- 记录生日、 anniversaries 及其他重要日期。
- 提供倒计时与提醒服务。

## 安装与使用

### 环境要求
- Flutter SDK >= 3.10.0
- Dart SDK >= 3.0.0
- 支持 Android /iOS 移动端运行

### 安装步骤
```bash
flutter pub get # 获取依赖
flutter run # 运行项目
flutter build apk # 生成 APK
flutter pub run flutter_launcher_icons # 生成图标
